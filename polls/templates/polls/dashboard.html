Now that the questionnaire is completed and we have a final percentage of Accountability, it is time to step into the realm of Explainability. We present to you, an experimental platform that consists of 3 machine learning models that take the user's input and make predictions about different things.Specifically the models are the following:
1)Miles per gallon prediction by a Random Forests algorithm
2)Diabetes prediction by a Logistic Regression algorithm
3)Iris classification by a k-Nearest Neighbors algorithm(k-NN)
After the prediction, we will explain to the user ,why and how the prediction was the outcome of the chosen model. We will explain the predictions with the use of SHAP(SHapley Additive exPlanations), which is a game theoretic approach able to explain the output of any machine learning model.
The following experimental platform aims to show how an organization or institute can use artificial intelligence, while explaining thoroughly how a prediction was made to the end-users, especially now that laws like GDPR are becoming more and more prominent.
