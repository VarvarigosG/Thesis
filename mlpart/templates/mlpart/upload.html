→Along with the prediction, we provide an explanation as to why the model came to the specific outcome. That is achieved with the help of SHAP. SHAP (SHapley Additive exPlanations) is a game theoretic approach to explain the output of any machine learning model. It connects optimal credit allocation with local explanations using the classic Shapley values from game theory and their related extensions.
